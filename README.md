# ClipShare Pro 📋

**Gerçek Zamanlı Paylaşım Platformu**  
**[clipshare.ghos.tr](https://clipshare.ghos.tr)**

Birden fazla cihaz arasında **anında** metin ve görüntü paylaşmanızı sağlayan modern ve hızlı bir araç.

![ClipShare Pro](https://i.ibb.co/yFCRDybp/mh71jsdb.png)

## ✨ Özellikler

- Gerçek zamanlı metin ve görüntü paylaşımı (Socket.io)
- Oda tabanlı sistem (kod ile katılma)
- Panodan yapıştır ve otomatik paylaş
- Görüntü yükleme + tek tıkla indirme
- Koyu / Aydınlık tema desteği
- PWA (Progressive Web App) - Telefona yüklenebilir
- Paylaşım geçmişi (Tarihçe)
- Tamamen responsive (Mobil + Masaüstü)

## 🚀 Hızlı Erişim

**🌐 Canlı Site:** [https://clipshare.ghos.tr](https://clipshare.ghos.tr)

## Nasıl Kullanılır?

1. Siteye girin
2. **"Yeni Oda Oluştur"** butonuna tıklayın
3. Oda adını belirleyin (isteğe bağlı şifre ekleyebilirsiniz)
4. Oluşan oda kodunu arkadaşlarınızla paylaşın
5. Metin yazın, panodan yapıştırın veya görüntü yükleyin
6. Tüm katılımcılarda **anında** görünür!

## 🛠 Kullanılan Teknolojiler

- **Frontend**: HTML, Tailwind CSS, Vanilla JavaScript
- **Backend**: Node.js + Socket.io
- **PWA**: Manifest + Service Worker
  
## 📱 PWA Yükleme

- Chrome/Edge'de adres çubuğunun sağ tarafında **Yükle** butonu çıkar
- Veya tarayıcı menüsünden **"Uygulamayı Yükle"** seçeneğini kullanın
